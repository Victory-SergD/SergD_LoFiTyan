## 📖 Что это

**LoFiTyan** — лёгкое настольное приложение (на Tauri), которое играет **настоящую lo-fi музыку из интернет-радио** ([radio-browser.info](https://www.radio-browser.info), станции lofi/chillhop) на фоне уютной сцены с персонажем. Поверх музыки можно подмешивать **атмосферу** (дождь, гром, лес, костёр), а управление при простое **само скрывается** — остаётся чистая «живая обоина». Работает на **macOS, Windows и Linux**.

> Музыка идёт из интернет-радио (нужна сеть); атмосфера, фон и интерфейс — локальные.

## 🗺️ Статус и дорожная карта

**✅ Готово (радио-фундамент + полировка + выбор станций):** реальное lo-fi радио (play/pause, ◀ ▶ переключение станций) с **устойчивой загрузкой** (перебор нескольких зеркал + повтор по тапу на ошибке), **выбор станций** (панель с жанрами Lo-Fi/Chillhop/Focus/Sleep, кураторский набор HQ-станций, избранное ★, вкладка «Ещё» из radio-browser, память последней станции), индикаторы загрузки/буферизации/ошибки, атмосфера поверх, мастер-громкость, фон/персонаж, авто-скрытие управления (с мгновенным возвратом — первый клик сразу попадает), **настоящий полноэкранный режим** (edge-to-edge, кнопка ⛶ / выход по `Esc`), честные горячие клавиши, настройки (громкость/фон/язык). В интерфейсе приложение представляется как **LoFiTyan**. Балласт удалённого генеративного движка вычищен (нет `tone`, ~26 МБ мёртвых ассетов). Проверено в реальном Tauri-окне.

## ✨ Возможности (сейчас)

- 🎵 Настоящее lo-fi радио: станции из radio-browser.info, play/pause, переключение ◀ ▶; устойчивая загрузка (перебор зеркал, повтор по тапу на ошибке).
- 🎛️ Выбор станций: панель с жанрами (Lo-Fi/Chillhop/Focus/Sleep), кураторский набор HQ-станций с бейджами битрейта, избранное ★, вкладка «Ещё» (radio-browser), память последней станции.
- ⏳ Понятная обратная связь: спиннер при загрузке/буферизации, сообщение при ошибке сети.
- 🌧️ Атмосфера поверх музыки: дождь, гром, лес, костёр (с визуалом дождя).
- 🔊 Мастер-громкость на радио и эффекты; стоп-всё по клавише `k`.
- 🖥️ Настоящий полноэкранный режим (edge-to-edge): кнопка ⛶ в доке, выход по `Esc`.
- ⌨️ Честные горячие клавиши: `Space` — play/pause радио, `K` — стоп-всё, `A/S/D/F` — эффекты, `←/→` — фон, `J` — настройки, `Ctrl/Cmd+I` — просторный режим.
- 🌌 «Просторный режим»: управление прячется при простое и **мгновенно** возвращается по движению — первый клик сразу попадает.
- 🖼️ Фон-сцена с персонажем; вертикальный/портретный режим.
- 🌍 7 языков интерфейса, включая русский.

## 🧩 Технологии

`Tauri 2` · `Svelte` · `TypeScript` · `Vite` · `radio-browser.info API` · `pnpm` · `Vitest`

## 🚀 Запуск локально

Понадобится: [Node.js](https://nodejs.org/), [pnpm](https://pnpm.io/), [Rust](https://www.rust-lang.org/) (stable) и [пререквизиты Tauri](https://tauri.app/start/prerequisites/) для твоей ОС.

```bash
git clone https://github.com/Victory-SergD/SergD_LoFiTyan
cd SergD_LoFiTyan
pnpm install
pnpm tauri:d   # запуск из исходников (режим разработки)
pnpm tauri:b   # сборка установщика (.dmg / .exe / .deb / ...)
```

Дополнительно: `pnpm dev` (только фронтенд), `pnpm test` (юнит-тесты, Vitest), `pnpm check` (проверка типов), `pnpm build`.

## 📂 Структура проекта

- `src/` — фронтенд на Svelte: интерфейс, компоненты, локали, сторы.
  - `src/lib/stores/radio.ts` — радио (radio-browser.info, перебор зеркал, кураторский seed + жанры + избранное + воспроизведение).
  - `src/lib/stores/` — также `volume`, `weather`, `immersion`, `fullscreen`, `picker`.
  - `src/lib/components/` — `RadioPlayer`, `StationPicker` (выбор станций), `Controls` (атмосфера + ⛶ полный экран), `Canvas` (фон), `Settings`, `TopBar`.
- `src-tauri/` — нативная оболочка (Rust / Tauri) и конфигурация (включая CSP под радио-стримы).
- `public/assets/` — фоны и звуки атмосферы.
- `docs/superpowers/` — документы дизайна, планы, QA и хэндофф-документ.
