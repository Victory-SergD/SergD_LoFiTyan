## ✅ Что сделано в этой итерации

Полный разворот: с **генеративного движка** на **реальное lofi-радио**.

- 🎵 **Радио** через [radio-browser.info](https://www.radio-browser.info): `loadStations`, play/pause, ◀ ▶ (next/prev), мастер-громкость на радио, стоп-всё (`k`). Store: `src/lib/stores/radio.ts` (фильтрует https-стримы под mixed-content, играет через единый `<audio>`).
- 🗑 **Удалено:** генеративный движок (`engine/`, Tone.js), аккорды, «New music», Auto-DJ, список «Track 1/2/3», старый визуализатор (полоски).
- 🌧 **Оставлено и работает:** атмосфера (дождь/гром/лес/огонь) поверх радио, фон/персонаж, **авто-скрытие** («просторный режим» — пользователю нравится), настройки (громкость, фон, язык).
- 🧪 **Качество:** 0 ошибок типов (было 10 — все были в удалённом движке), 49 юнит-тестов зелёные, CSP под API + стримы + favicon.
- ✔️ **Проверено:** радио реально играет в Tauri-окне (играл «0R - LO-FI», MP3 192k); атмосфера, громкость, стоп-всё, авто-скрытие — ок.

## 🎨 Персонаж-визуализатор — находки (под идею пользователя)

Готового drop-in «аудио-реактивного lofi-аватара» **нет**. Практичный путь — **анимированный ассет** персонажа (GIF/WebM/sprite/Lottie), сменяемый (у нас уже есть система фонов в настройках):

- [Lofi Girl Character Generator](https://www.lofigirl.com/generator) — официальный генератор персонажей (проверить лицензию на использование).
- [Animated Girl for Lo-Fi (itch.io)](https://slavarbuz.itch.io/animated-girl-for-lo-fi) — готовая анимация-ассет.
- [MoeWalls — lofi live wallpapers](https://moewalls.com/tag/lofi/) / [MotionBGs](https://motionbgs.com/tag:lofi/) — анимированные lofi WebM/GIF (живые обои).
- [VRoid Studio](https://vroid.com) / [CharacterStudio (VRM)](https://github.com/M3-org/CharacterStudio) — для своих аниме-персонажей 3D (тяжелее, на будущее, если захотим уникального).

⚠️ Аудио-реактивность для **радио** ограничена: кросс-ориджин стрим нельзя проанализировать через WebAudio (taint) → реальные «полоски под музыку» с радио не сделать. Проще: зацикленная анимация персонажа + опционально лёгкий generic-бит. Обдумать на следующей итерации.
