## Что закрыто (30 багов аудита + новое)
- **Звук/состояние:** единые сторы (volume/autodj/atmosphere/immersion); переключение «атмосферы» без задвоения; `await Tone.start()`; громкость без поллинга (все 4 эффекта на сторе); WORLD Авто-DJ через стор; кнопка «Новая музыка». *(audio-1..7, state-2,4,6,7, ui-1)*
- **Вертикальный режим:** окно тянется (портретные дефолты 480×900, мин 360×560); слой Canvas (фон+визуализатор+слот); раскладка в колонку и прокрутка в портрете. *(layout-1,2,3,5,7,8)*
- **Просторный режим:** управление само скрывается по простою (~3с), возвращается по движению; кнопка + хоткей Ctrl/Cmd+I; Canvas остаётся видимым.
- **Кнопки/UX:** эффекты подсвечиваются только при реальном звуке; кнопки окна (mac + win/linux) реагируют с первого клика и отражают состояние; хоткеи не срабатывают при вводе; настройки не моргают; «глаз» — настоящий тумблер. *(ui-2..8)*
- **Бэкенд/безопасность:** shell-плагин + грейсфул-старт; внешние ссылки в системном браузере; CSP включён; направление текста из локали. *(backend-1..6, state-6)*
- **Честный UX:** «Tracks» → «Атмосфера» (en/ru).

## Ручная проверка (рекомендуется пройти в живом приложении)
Запусти `pnpm tauri:d` и проверь:
1. Музыка стартует с первого клика и с пробела; «Новая музыка» (🔄) меняет аккорды.
2. Слои «Атмосферы» (1–9 / клики) не задваиваются; `k` останавливает всё.
3. Перетаскивание ползунков громкости меняет звук вживую.
4. Сделай окно узким/высоким → раскладка перестраивается в колонку, ничего не обрезано.
5. Не трогай мышь ~3с → управление прячется; двинь мышь → возвращается; Ctrl/Cmd+I.
6. Кнопки окна (закрыть/свернуть/развернуть) реагируют с первого клика.
7. Ссылка GitHub в инфо открывается в браузере.
